## 1. Sobre o Projeto

O Cão de Estrelas é um pet shop tradicional em Moema, com mais de **40 anos de atuação** no bairro e nota **4.9 no Google** com 249 avaliações reais.

Este projeto consiste em um site institucional de alta conversão com **duas frentes principais**:

- **Presença digital completa** com 32 páginas segmentadas por bairro, serviço e raça
- **Simulador de orçamento** integrado ao WhatsApp para conversão imediata

🔗 **Visualizar:** https://niamwite.github.io/caodeestrelas/

## 3. Simulador de Serviços — Conversão Instantânea

### O que é

Um simulador de orçamento em **6 passos** integrado diretamente ao WhatsApp do pet shop. O cliente monta seu pacote de serviços e envia o resumo sem sair do site.

### Passo a passo

```
PASSO 1 → Selecionar serviços       (banho, tosa, hidratação, spa, veterinário, hotel)
PASSO 2 → Escolher porte            (pequeno / médio / grande — multiplicador de preço)
PASSO 3 → Adicionar extras          (corte de unhas, limpeza de ouvidos, dental, perfumação)
PASSO 4 → Informar dados             (nome do tutor, nome do pet)
PASSO 5 → Escolher data             (calendário + observações)
PASSO 6 → Revisar e enviar          (resumo visual + botão WhatsApp)
```

### Recursos

- **Resumo em tempo real** — valores atualizam a cada seleção
- **Precificação automática** — porte multiplica o valor base
- **Integração WhatsApp** — mensagem formatada com todos os dados
- **Experiência mobile-first** — funciona perfeitamente em smartphones
- **Schema LocalBusiness** — autoridade local para SEO

### Exemplo de mensagem enviada ao WhatsApp

```
🐾 ORÇAMENTO — Cão de Estrelas Pet Shop

📋 Serviços: Banho + Tosa Higiênica
🐕 Porte: Médio (+25%)
✨ Extras: Corte de unhas + Perfume

💰 Valor estimado: R$ 142,50

👤 Tutor: Maria Silva
🐶 Pet: Thor
📅 Data: 22/04/2025
📝 Obs: Thor é ansioso, Avisar antes

➡️ Aguardamos você no pet shop!
```

### Por que funciona

- **Zero atrito** — o cliente não precisa ligar ou ir pessoalmente
- **Percepção de valor** — o resumo visual justifica o preço
- **Escalabilidade** — funciona 24h sem intervenção humana
- **Dados qualificados** — o pet shop recebe nome, pet e intenção clara

## 4. Arquitetura de SEO — Dominação Local

### Estratégia de palavras-chave

O site targeting três tipos de busca:

```
TIPO 1 — Busca local pura
"pet shop moema"
"veterinario moema"
"hotel pet moema"

TIPO 2 — Busca local + serviço
"banho e tosa moema"
"tosa schnauzer moema"
"creche pet moema"

TIPO 3 — Busca local + bairro vizinho
"pet shop vila mariana"
"pet shop ibirapuera"
"pet shop indianopolis"
```

### O que cada página inclui

| Elemento SEO | Implementação |
|--------------|--------------|
| `<title>` otimizado | Keyword + local + nome da marca |
| `<meta description>` | Unique, com CTA, 150-160 caracteres |
| `<canonical>` | URL oficial para evitar conteúdo duplicado |
| Open Graph tags | Facebook/WhatsApp sharing otimizado |
| Schema.org LocalBusiness | Dados estruturados para Google |
| Schema.org FAQPage | Rich snippets no Google |
| `<h1>` com keyword | Uma vez por página, destacado visualmente |
| Links internos | 4-8 links por página para páginas relacionadas |
| `<meta robots>` | index,follow em todas as páginas |
| Sitemap XML | 32 URLs submetidas ao Google |

### FAQ com Rich Snippet

Cada página tem 3 FAQs com Schema FAQPage — essas aparecem como **perguntas e respostas diretamente no Google**, aumentando o CTR.

```
Exemplo de exibição no Google:

─────────────────────────────────
Veterinário em Moema | Cão de Estrelas
niamwite.github.io/caodeestrelas/veterinario-moema
▶ Tem veterinário no pet shop?

veterinária com avaliação completa em Moema.
▶ Quanto custa consulta veterinária em Moema?
A partir de R$ 120, com avaliação clínica...
▶ Vocês aplicam vacinas?
Sim. Vacinação completa com cartão...
─────────────────────────────────
```
